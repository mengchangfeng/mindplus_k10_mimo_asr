# K10 MiMo语音识别与合成 Mind+扩展

面向行空板 K10 的 ArduinoC Mind+ 用户扩展，封装语音识别与非流式语音合成：

- 初始化 MiMo API Key 和音色
- 开始录音
- 结束录音并返回识别结果
- MiMo 合成语音（文本、风格）

底层使用 K10 板载 ES7243E/I2S 麦克风采集 16kHz/16bit 单声道 WAV，并调用 `mimo-v2.5-asr` 识别；语音合成固定使用 `mimo-v2.5-tts`，返回 24kHz/16bit 单声道 WAV 后通过 K10 扬声器播放。

## 使用

1. Mind+ 切换到上传模式，选择 K10/ESP32-S3 对应主控。
2. 扩展中本地导入本目录的 `config.json`。
3. 先用 Mind+ 自带 WiFi 积木连接网络。
4. 调用“初始化 MiMo”，填入 API Key 和音色（例如“冰糖”）。API 地址和模型已固定为 MiMo 官方接口。
5. 调用“开始录音”开始后台录音。
6. 调用“结束录音并返回识别结果”停止录音并获得识别文本。

录音最长时长固定为 5 秒。语音合成积木的风格参数可填写“温柔、缓慢”等自然语言，留空使用默认自然语气。

## 变更

- `0.0.5`：增加 MiMo 非流式语音合成；固定 API 地址和模型，初始化只保留 API Key 与音色；增加每次请求后的音频、Base64、HTTP 和 I2S 资源清理。
