# K10 MiMo语音识别 Mind+扩展

面向行空板 K10 的 ArduinoC Mind+ 用户扩展，封装三条积木：

- 初始化设置 API 地址、API Key、模型和最长录音秒数
- 开始录音
- 结束录音并返回识别结果

底层使用 K10 板载 ES7243E/I2S 麦克风采集 16kHz/16bit 单声道 WAV，并调用 `mimo-v2.5-asr` 识别。

## 使用

1. Mind+ 切换到上传模式，选择 K10/ESP32-S3 对应主控。
2. 扩展中本地导入本目录的 `config.json`。
3. 先用 Mind+ 自带 WiFi 积木连接网络。
4. 调用“初始化设置”，填入 API 地址、MiMo API Key、模型和最长录音秒数。
5. 调用“开始录音”开始后台录音。
6. 调用“结束录音并返回识别结果”停止录音并获得识别文本。

录音最长时长由初始化积木设置，建议 3-5 秒。
